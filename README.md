### package.json
```json
{
  "scripts": {
    "template": "node node_modules/taro-template/template.js"
  }
}
```

### Usage

- 创建页面(同步创建scss和interface)
```
yarn template p directoryName fileName
```
- 创建model
```
yarn template m directoryName fileName
```
- 创建service
```
yarn template s directoryName fileName
```
